### Nerd Guide
- Clone this repository and make sure you have a local LaTeX compiler.
- Go to the working directory of your tex file and run `make`. For example, to generate Extra Practice 1, run
    ```sh
    cd ~/practice-makes-perfect/01
    make
    ```
